=head1 stackato-apps

This repo contains all the latest Stackato Apps in one repository that you can
clone with this command:

    git clone https://github.com/Stackato-Apps/stackato-apps/

The apps are all under the C<apps/> directory and they are subrepo clones of
the various Stackato-Apps repositories.

There are also tagging symlink directories like C<tags/java>.

=head2 Updating this Repo

If you are an Activator, to update this repo to all the latest app versions:

      # If you don't have git-subrepo:
      git clone https://github.com/ingydotnet/git-subrepo
      . $PWD/git-subrepo/init

      cd stackato-apps
      git subrepo pull --all
      git push
